## ğŸ¨ Customization Guide

### Personal Information
Update the following sections in `index.html`:

1. **Hero Section** (Lines 60-80):
   - Your name and title
   - Professional description
   - Social media links

2. **About Section** (Lines 100-150):
   - Personal description
   - Statistics (projects, experience, satisfaction)
   - Technical skills and proficiency levels

3. **Experience Section** (Lines 160-220):
   - Work history
   - Job descriptions
   - Technologies used

4. **Projects Section** (Lines 230-350):
   - Project details
   - Screenshots
   - Technology stacks
   - Live demo and GitHub links

5. **Education Section** (Lines 360-420):
   - Degrees and certifications
   - Institutions and dates
   - Achievements

6. **Contact Section** (Lines 430-500):
   - Contact information
   - Location
   - Social media links

### Styling
Customize colors and appearance in `styles.css`:

- **Colors**: Modify CSS custom properties at the top of `styles.css`
- **Fonts**: Change the font family in the `:root` section
- **Spacing**: Adjust spacing variables for different layouts
- **Animations**: Modify animation duration and easing

### Adding New Sections
To add new sections:

1. Add HTML structure in `index.html`
2. Add corresponding navigation link
3. Style the section in `styles.css`
4. Add any interactive functionality in `script.js`
